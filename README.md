README

Environment specs:
PHP 7.4.33(cli) 
MYSQL 5.7 
Laravel 8
Vue 3

Steps to setup on local:
1. npm install
2. composer install
3. create a .env file from the sample provided

steps to run the project:
1. run 
npm run build
2. run
npm run dev
3. in another terminal window (but within project directory) run
php artisan serve
